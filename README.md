# SOC Log Analysis Project

## Overview
This project demonstrates SOC (Security Operations Center) skills through log analysis of authentication and web server logs to detect suspicious activity and brute-force attacks.

## Key Features
- Manual log analysis (SSH & web logs)
- Detection of suspicious IP activity
- Incident report with findings and timeline
- Python script for automated detection

## Project Structure
- logs/
- report/
- scripts/
- screenshots/

## Automation
A Python script was developed to detect suspicious IPs based on repeated failed login attempts.
